# Sort Command

| :[range]sort[!]  	| Description	|
| ------------: | ---------- |
| **:sort** 		| sort lines Alphabetically |
| **:[range]sort** 		| define the range of the ordenation, use ',' to specify the range of the lines |
| **sort!** 		| define the reverse sort |
